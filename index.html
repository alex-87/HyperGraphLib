Welcome to HyperGraphLib's page.

HyperGraphLib is a C++ library which allows you to work with hypergraphs (a graphe which edges can contains more than 2 vertexes) using the HyperGraphLib in your C++ app. Easily :)

Implemented algorithms

With this library, you can work with those algorithms:

  • Connected
  • Dual
  • Helly
  • k-Regular
  • k-Uniform
  • Linear
  • Simple

Some of them only decide of the associated property.

Application instead of library

With the client package (included by default), compile it with:

Compilation

cmake CMakeLists.txt && make

Try it :)

How to generate a random hypergraph with 5 vertexes ?

./HypergraphLib --random 5 > myAwesomeHypergraph.txt

How to get the adjacent matrix ?

./HypergraphLib --inputfile myAwesomeHypergraph.txt --adjacence

How to know if my hypergraph is connected (each vertex can be reach) ?

./HypergraphLib --inputfile myAwesomeHypergraph.txt --connexe

How to use HyperGraphLib ?

./HypergraphLib --help
